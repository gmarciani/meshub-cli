Installation
============

Install the Meshub CLI using pip:

.. code-block:: bash

   pip install meshub

Or install from source:

.. code-block:: bash

   git clone https://github.com/gmarciani/meshub-cli.git
   cd meshub-cli
   pip install -e .

Configuration
-------------

After installation, configure your API key:

.. code-block:: bash

   meshub config set ApiKey YOUR_API_KEY

Verify the installation:

.. code-block:: bash

   meshub --version
